# Code of Conduct

## Our Commitment

This project is committed to a respectful, inclusive, and harassment-free experience for everyone,
regardless of background, identity, experience level, or role.

## Expected Behavior

- Be respectful and constructive in issues, pull requests, and reviews.
- Assume good intent and focus feedback on ideas, not people.
- Keep discussions technical, clear, and professional.
- Accept and act on maintainers' moderation decisions.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks, insults, or trolling.
- Publishing private information about others without explicit permission.
- Repeatedly derailing technical discussions.

## Scope

This code of conduct applies in all repository spaces, including issues, pull requests, reviews, and
other project communications.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct. They may remove,
edit, or reject contributions and comments that violate these standards.

## Reporting

To report violations, open a private security report if sensitive details are involved, or contact the
repository owner via GitHub profile/contact channels.

## Consequences

Maintainers may apply corrective actions proportionate to behavior, including warning, temporary
restriction, or permanent ban from participation in this project.
